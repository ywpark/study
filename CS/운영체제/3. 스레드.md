# 스레드

---

## 기본 정보

- 프로세스를 구성하는 실행 흐름의 단위
- 하나의 프로세스는 하나 이상의 스레드를 가질 수 있음
- 스레드의 구성 요소
    - 실행에 필요한 최소한의 정보
        - 스레드ID
        - 프로그램 카운터 및 레지스터 값
        - 스택
- 모든 스레드들은 프로세스의 자원을 공유

## 멀티 프로세스 VS 멀티 스레드

- `멀티 프로세스`
    - 기본 정책은 모든 자원이 별도로 구성 ( 공유 안됨 )
    - 자원을 공유할 수 있는 방법이 존재
        - 단, 비용이 크다
        - `프로세스 간 통신 (IPC)`
            - `파일을 통한 프로세스 간 통신`
            - `공유 메모리를 통한 프로세스 간 통신`
                - `POSIX shared memory`
- `멀티 스레드`
    - 모든 자원이 공유
